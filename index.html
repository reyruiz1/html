                            <head>
                               <meta charset="UTF-8">
                               <meta name="viewport" content="width=device-width, initial-scale=1.0">
                               <title>Document</title>
                               <link rel="stylesheet" href="estilos.css">

                            <link href="/media/examples/link-element-example.css" rel="stylesheet" />

<Meta> Es una etiqueta usada para proporcionar información adicional sobre el documento web. Esta información incluye detalles como el conjunto de caracteres, descripción, palabras clave y configuraciones de visualización, ayudando a los navegadores y motores de búsqueda a interpretar y mostrar adecuadamente el contenido. <meta charset="UTF-8"> Más información
<style> Es una etiqueta en HTML utilizada para incrustar reglas de estilo o estilos CSS (Cascading Style Sheets) directamente en un documento HTML. Permite definir la apariencia y el diseño de elementos HTML, como colores, fuentes, márgenes, etc., sin necesidad de un archivo externo de estilo. Esto facilita la personalización y el diseño visual de la página web. 
                            <style>
                                /* Definición de estilos CSS */
                                body {
                                  font-family: Arial, sans-serif;
                                  background-color: #f0f0f0;
                                }
                            
                                h1 {
                                  color: blue;
                                }
                            
                                p {
                                  font-size: 18px;
                                }
                              </style>

<title> Utilizada para definir el título del documento web. Este título aparece en la barra de título del navegador y proporciona una breve descripción del contenido de la página. Es una parte importante de la estructura de un documento HTML, ya que ayuda a los usuarios y motores de búsqueda a comprender el tema o la naturaleza de la página. 
                            <!DOCTYPE html>
                            <html>
                            <head>
                               <title>Mi Página Web</title>
                            </head>
                             <body>
                               <h1>Bienvenido a mi página web</h1>
                               <p>Esta es una página de ejemplo.</p>
                             </body>
                            </html>

<aside> es una etiqueta utilizada para representar contenido relacionado o auxiliar que está apartado del contenido principal de la página, como anuncios, enlaces relacionados, biografías, o cualquier contenido adicional. 
                            <!DOCTYPE html>
                            <html>
                            <head>
                              <title>Mi Página Web</title>
                            </head>
                            <body>
                              <h1>Bienvenido a mi página web</h1>
                              
                              <article>
                                <h2>Artículo principal</h2>
                                <p>Este es el contenido principal de mi artículo.</p>
                              </article>
                            
                              <aside>
                                <h2>Contenido relacionado</h2>
                                <ul>
                                  <li><a href="#">Enlace 1</a></li>
                                  <li><a href="#">Enlace 2</a></li>
                                  <li><a href="#">Enlace 3</a></li>
                                </ul>
                              </aside>
                            </body>
                            </html>

<h1> , <h2>, <h3> , <h4> , <h5> , <h6> Son utilizadas para definir títulos y subtítulos en una página web. Cada etiqueta representa un nivel de encabezado. 
                            <!DOCTYPE html>
                            <html>
                            <head>
                              <title>Ejemplo de Títulos en HTML</title>
                            </head>
                            <body>
                            
                              <h1>Título de nivel 1</h1>
                              <p>Este es un contenido relacionado con el título principal.</p>
                            
                              <h2>Título de nivel 2</h2>
                              <p>Este es un contenido relacionado con un subtítulo.</p>
                            
                              <h3>Título de nivel 3</h3>
                              <p>Este es un contenido relacionado con otro nivel de subtítulo.</p>
                            
                              <h4>Título de nivel 4</h4>
                              <p>Contenido relacionado con un nivel más bajo.</p>
                            
                              <h5>Título de nivel 5</h5>
                              <p>Contenido relacionado con un nivel aún más bajo.</p>
                            
                              <h6>Título de nivel 6</h6>
                              <p>Contenido relacionado con el título de menor nivel.</p>
                            
                            </body>
                            </html>
